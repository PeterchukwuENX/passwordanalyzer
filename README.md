 Password Analyzer

 Overview
A Python-based password strength analyzer that evaluates user passwords based on security rules.  
It checks for:

- Minimum length (12 characters)  
- Presence of at least one digit  
- Presence of uppercase and lowercase letters  
- Presence of special characters  

The program scores passwords and outputs a **Weak / Medium / Excellent** verdict.

 Features
- Modular design using functions
- Clear, interactive console output
- Phase 1 Python fundamentals: loops, conditionals, flags, functions
- Ready for extension into more advanced security tools
